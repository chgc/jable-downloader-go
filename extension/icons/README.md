# Extension Icons (可選)

圖標是可選的，Chrome 會使用默認圖標。

如果想自定義圖標，請將以下尺寸的圖標放置在此目錄：
- icon16.png (16x16)
- icon48.png (48x48)  
- icon128.png (128x128)

然後在 manifest.json 中取消註釋 icons 配置。

## 快速生成圖標

使用在線工具（無需本地處理）：
- https://www.favicon-generator.org/
- https://icon.kitchen/
- https://realfavicongenerator.net/

建議：下載圖標，紫色漸變背景 + 白色下載箭頭
